# SQL Data Analysis Project

## Objective

Analyze an e-commerce orders dataset using SQL and extract meaningful business insights.

## Skills Used

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* COUNT()
* SUM()
* AVG()

## Dataset

Orders dataset containing customer, product, payment, and revenue information.

## Key Insights

* Calculated total orders.
* Calculated total revenue.
* Found average order value.
* Analyzed revenue by payment method.
* Analyzed orders by status.

## Tools Used

* SQLite
* DB Browser for SQLite
* Git & GitHub
