# Data Visualization Project

## Overview
This project focuses on visualizing data using Python libraries such as Pandas and Matplotlib. The goal is to convert raw data into meaningful charts and graphs to identify trends and insights.

## Dataset
The dataset contains information related to:
- Order Status
- Payment Method
- Order Date
- Total Price

## Visualizations Created

### 1. Order Status Distribution (Bar Chart)
Displays the number of orders in each status category such as Delivered, Shipped, Pending, Cancelled, and Returned.

### 2. Payment Method Distribution (Pie Chart)
Shows the percentage share of different payment methods used by customers.

### 3. Monthly Sales Trend (Line Chart)
Illustrates how sales change over time and helps identify growth patterns and seasonal trends.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook / VS Code

## Key Insights
- Online payment methods are widely used.
- Sales show variation across different months.
- Order statuses are distributed across multiple categories, helping analyze business performance.
