# Project 2: Exploratory Data Analysis (EDA)

## Objective

Analyze a dataset to understand patterns, trends, distributions, and key insights.

## Basic Statistics

* Total Records: 1200
* Total Columns: 14
* Average Quantity Ordered: 2.95
* Average Unit Price: ₹356.41
* Average Items in Cart: 5.49
* Average Total Price: ₹1053.97

## Median Values

* Quantity: 3
* Unit Price: ₹364.21
* Items in Cart: 5
* Total Price: ₹823.62

## Data Quality

* CouponCode column contains 309 missing values.
* All other columns are complete.

## Outlier Analysis

* 8 outliers were detected in the TotalPrice column using the IQR method.
* These outliers represent unusually high or low-value orders.

## Key Observations

1. Customers purchase approximately 3 items per order on average.
2. The average order value is ₹1053.97.
3. Most orders fall within the normal price range, with only a few extreme values.
4. Missing values are present only in the CouponCode column.
5. Customer purchasing behavior appears relatively consistent across the dataset.

## Conclusion

The dataset was analyzed to understand its structure, distribution, and purchasing patterns. Basic statistical measures such as mean, median, and count were calculated for numerical attributes. The analysis revealed that customers purchase approximately 3 items per order on average and spend around ₹1054 per transaction.

The dataset contains 1200 records and 14 attributes. A total of 309 missing values were identified in the CouponCode column, while all other columns were complete. Outlier detection using the IQR method identified 8 outliers in the TotalPrice attribute.

Overall, the dataset provides useful insights into customer purchasing behavior and is suitable for further business intelligence and predictive analytics tasks.
